# Installing spacectl

spacectl은 command line 으로 SpaceONE을 제어할 수 있게 하는 Tool 입니다. 

## Prerequisite

설치를 위해서는 python 3.8 이상이 설치된 환경이 필요 합니다. 



## Package 설치하기

{% tabs %}
{% tab title="Mac OS" %}
```text
# Installing spacectl & dependency
pip3 install spaceone-core spaceone-api spacectl
pip3 list|grep space
```
